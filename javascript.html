1. JavaScript Basics

JavaScript is a dynamic, client-side scripting language for web interactivity.

console.log("Hello, World!");
var x = 10;           // Older variable declaration
let y = 20;           // Block-scoped
const z = 30;         // Constant

Detailed Explanation:

  • var: Function-scoped, hoisted
  • let: Block-scoped, no hoisting
  • const: Block-scoped, immutable value
  • Types: Number, String, Boolean, etc.
Prefer let and const over var.

2. Functions

Functions encapsulate reusable code.

function greet(name) {
    return "Hello, " + name;
}

const add = (a, b) => a + b; // Arrow function

console.log(greet("Alice")); // "Hello, Alice"
console.log(add(2, 3));      // 5

Features:

  • Declaration: function keyword
  • Arrow: Concise syntax, no own this
  • Scope: Global, function, block

3. Objects

Objects store key-value pairs.

const person = {
    name: "John",
    age: 30,
    greet() {
        return "Hi, I'm " + this.name;
    }
};

console.log(person.name);    // "John"
console.log(person.greet()); // "Hi, I'm John"

Features:

  • Properties: Data storage
  • Methods: Functions in objects
  • this: Refers to current object

4. Arrays

Arrays store ordered lists.

const fruits = ["apple", "banana"];
fruits.push("orange");       // Add to end
fruits.pop();               // Remove from end

fruits.forEach(fruit => console.log(fruit));

Methods:

  • push/pop: Add/remove at end
  • shift/unshift: Start operations
  • forEach/map: Iteration

5. Control Flow

Controls execution with conditions and loops.

let x = 5;
if (x > 0) {
    console.log("Positive");
}

for (let i = 0; i < 3; i++) {
    console.log(i); // 0, 1, 2
}

Structures:

  • if-else: Conditional
  • for/while: Loops
  • switch: Multi-case

6. DOM Manipulation

Interact with HTML elements.

const element = document.getElementById("myId");
element.textContent = "New text";
element.style.color = "blue";

Features:

  • getElementById: Select by ID
  • querySelector: CSS selector
  • Properties: textContent, style
DOM links JS to HTML/CSS.

7. Events

Handle user interactions.

const button = document.querySelector("button");
button.addEventListener("click", () => {
    console.log("Clicked!");
});

Features:

  • addEventListener: Attach handlers
  • Events: click, mouseover, keydown
  • Event Object: Details of event

8. Asynchronous JavaScript

Handle operations that take time.

fetch("https://api.example.com/data")
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error(error));

async function getData() {
    const response = await fetch("https://api.example.com/data");
    const data = await response.json();
    console.log(data);
}

Features:

  • Promises: then/catch
  • async/await: Cleaner syntax
  • Callbacks: Older approach

9. ES6 Features

Modern JavaScript enhancements.

let x = 10;              // Block scope
const y = 20;            // Constant
const add = (a, b) => a + b; // Arrow function
const [a, b] = [1, 2];   // Destructuring

Features:

  • let/const: Better scoping
  • Arrow Functions: Concise syntax
  • Destructuring: Extract values

10. Error Handling

Manage runtime errors.

try {
    let result = riskyOperation();
} catch (error) {
    console.error("Error:", error.message);
} finally {
    console.log("Done");
}

Features:

  • try-catch: Handle errors
  • finally: Always runs
  • throw: Custom errors
