2. Three Ways to Add CSS in HTML

  1. Inline CSS
  2. Internal CSS
  3. External CSS

Inline CSS

The code is written in the opening tag of the HTML element by using style attribute.

Syntax:

<p style="property : value"> content </p>

Internal CSS

The code is written inside or after head tag section of an HTML element by using <style> tag.

Syntax:

<head>
    <style>
        tag_name or selector {
            property: value;
        }
    </style>
</head>

External CSS

External CSS styling can be done by creating an external CSS file with file_name.css extension and providing the link between that CSS file to HTML file by <link> tag in the head section of HTML element.

Syntax:

<head>
    <link rel="stylesheet" href="file_name.css">
</head>

NOTE:

rel: specifies the relation. rel="stylesheet" No value other than "stylesheet" should be declared

3. CSS Selectors

Selectors are used to select or target the particular HTML element to style them. Or it is term or word which we will mention to style multiple elements at the same time

There are five types of Selectors:

  1. Simple Selectors.
  2. Combinator Selectors.
  3. Attribute Selectors.
  4. Pseudo Class Selectors.
  5. Pseudo Element Selectors.

Simple Selector

Simple selectors style the HTML element in five ways:

  1. Id Selector (#)
  2. Class Selector (. (dot))
  3. Tag name selector (tagname itself)
  4. Grouping selector (, (comma))
  5. Universal selector (*)

ID Selector

Id selector targets only individual element in the HTML document.

Prefix symbol used for id selector is # (hash) id should be unique

HTML file:

<p id="demo">HELLO</p>

CSS file:

#demo {
    color: red;
    background: yellow;
}

Class Selector

Class selectors targets multiple elements in the HTML document.

Prefix symbol used for id selector is . (dot) class can be multiples

Syntax:

HTML:

<h1 class="test">HELLO</h1>
<h2 class="test">GOOD</h2>
<h3 class="test">MORNING</h3>

CSS:

.test {
    color: magenta;
    background: green;
}

Tagname Selector

Tagname selector targets the HTML elements by tag name.

There is no symbol used in tagname selectors. We have to use tag as name itself

Syntax:

<h1>HEADING 1</h1>
<h2>HEADING 2</h2>
<h3>HEADING 3</h3>
        
h1 {
    color: magenta;
    background: green;
}
h2 {
    color: red;
    background: black;
}
h3 {
    color: yellow;
    background: orange;
}

Grouping Selector

Grouping selector targets a group of HTML elements by tagname separated by commas (,)

Syntax:

<p>Paragraph</p>
<div>Division</div>
<h4>HEADING 4</h4>
        
p, div, h4 {
    color: magenta;
    background: green;
}

Universal Selectors

Universal Selectors targets every HTML element. The symbol of Universal Selector is *

Syntax:

<p>Paragraph</p>
<div>Division</div>
<h4>HEADING 4</h4>
        
* {
    color: yellow;
    background: magenta;
}

Combinator Selector

Combinator Selectors are used to style the combination of HTML elements.

  1. Descendent Selector: This Selector targets both direct and indirect children of a parent tag. The symbol is (space)
    2. Syntax: parent_tag child_tag { property: value; }
  2. Child Selector: This Selector targets only the direct children of the parent. The symbol is > (greater than)
    3. Syntax: parent_tag > child_tag { property: value; }
  3. Adjacent Selector: This Selector targets the element which is the first sibling of the targeted HTML tag or element. (or) This Selector targets the element which is exactly adjacent to the targeted HTML tag or element. The symbol is + (plus)
    4. Syntax: first_sib_tag + second_sib_tag { property: value; }
  4. General Sibling Selector: This Selector targets all the siblings of the target HTML tag or element. (or) This Selector targets all the adjacent tags of the target HTML tag or element. The symbol is ~ (tilde)
    5. Syntax: first_sib_tag ~ second_sib_tag { property: value; }

Attribute Selectors

Attribute selectors are used to style the HTML element by targeting the respective attributes and with their values.

Types of Attribute Selector:

  1. tag_name[attribute_name]: Represents elements with an attribute name of attribute_name.
  2. tag_name[attribute_name="value"]: Represents element with an attribute name of attribute_name whose value is exactly value.

Pseudo Elements Selectors

For pseudo elements selectors will target by (::) this symbol

  1. First Letter: It will target the first letter of respective content of targeted HTML element
  2. First Line: It will target the first line of respective content of targeted HTML element
  3. Before: It will add the any type of content or CSS property just before the targeted HTML element
  4. After: It will add the any type of content or CSS property right after the targeted HTML element
  5. Selection: Whenever we will select any kind of content will get color as blue as default but by using selection selector we can change that color and background color to
  6. Marker: This selector will change the color of list marker and only it will applicable for li tag

4. Box Modelling

Box Model is essentially a box that wraps around every HTML element

Box Model is used to design and layout.

The CSS Box Model consists of Margin, Border, Padding and the actual Content.

  • Content: It is the content of the HTML element.
  • Padding: It is the space after the content.
  • Border: It is the Border of the HTML element.
  • Margin: It is the space after the Border of an HTML element.

Padding and Margin allow up to four values.

Syntax:

|  | T | R | B | L |
|------|-----|-----|-----|
| padding: 10px | 0 | 0 | 0 |
| padding: 10px 20px | 0 | 0 |
| padding: 10px 20px 30px | 0 |
| padding: 10px 20px 30px 40px |

8. CSS Background Properties

background color:

To add background color in HTML, use the CSS background-color property. Set it to the color name or code you want and place it

Background:

The only background property we will use for adding mix-up of two or more colors in 3 ways:

  • Linear-gradient: It adds two or more colors by horizontally
  • Radial-gradient: It add two or more colors from middle point to around all rounded surface
  • Conic-gradient: It will add two or more colors from center pointer of HTML element in shape or triangle like ex. CD DVD shape

background image:

The most basic way to set a full-page background is to use the CSS background-image property. This property accepts a value that is the URL of the image you want to use as the background.

Background size:

We can set the background image size with the help of this property. The background-size property is specified in one of the following ways:

  • Using the keyword values contain or cover.
  • Using a width value only, in which case the height defaults to auto.
  • Using both a width and a height value, in which case the first sets the width and the second sets the height.

background position:

The background-position property sets the starting position of a background image. Tip: By default, a background-image is placed at the top-left corner of an element

background attachment:

The background-attachment property sets whether a background image scrolls with the rest of the page, or is fixed. Values are scroll and fixed

background repeat:

The background-repeat property sets if/how a background image will be repeated. By default, a background-image is repeated both vertically and horizontally. Tip: The background image is placed according to the background-position property.

Values: repeat, no-repeat, repeatX, repeatY

Filter Properties:

The filter property defines visual effects (like blur and saturation) to an element (often <img>).

filter: hue-rotate(50deg);
filter: brightness(70%);
filter: drop-shadow(10px 10px 10px red);
filter: grayscale(100%);
filter: saturate(30%);
filter: opacity(100%);
filter: sepia(30%);
filter: blur(10px);

Box Shadow

The box-shadow CSS property adds shadow effects around an element's frame.

box-shadow: 5px 5px 5px 5px color;

  • 5px 5px - Specify a Horizontal and a Vertical Shadow. In its simplest use, you only specify a horizontal and a vertical shadow
  • 5px - Add a Blur Effect to the Shadow
  • 5px - Set the Spread Radius opacity of the Shadow
  • Color - Specify a Color for the Shadow. The color parameter defines the color of the shadow.

Overflow Property

The overflow property specifies what should happen if content overflows an element's box.

This property specifies whether to clip content or to add scrollbars when an element's content is too big to fit in a specified area.

Values --- hidden, scroll

Resize Property

The resize property defines if (and how) an element is resizable by the user.

Note: The resize property does not apply to inline elements where overflow="visible(scroll)". So, make sure that overflow is set to "scroll", "auto", or "hidden".

Values: horizontal, vertical, both

Visibility Property

This property makes the content visible or hidden according to its values. The values are given in visibility properties are visible and hidden

10. Positions in CSS

Relative: The position of the element is relative to the normal position, where the white space of the targeted element is not occupied by the other element

Absolute: When the position is absolute the element can be placed anywhere on the webpages, where the white space of that element is occupied by the other element.

Fixed: The position of the element depends on the viewport of the browser's Window and it will be fixed on the webpages

Sticky: The targeted element is sticked to the position based on the scrolling of the web page.

Static: Default position for HTML element

11. Display Properties

Inline: Displays an element as an inline element (like <span>). Any height and width properties will have no effect. This is default.

Block: Displays an element as a block element (like <p>). It starts on a new line, and takes up the whole width

Inline-block: Displays an element as an inline-level block container. The element itself is formatted as an inline element, but you can apply height and width values

12. Flex Property

The most flexible display property in CSS is the apparently named as display: flex, which allows you to create flexible layouts with elements that you can align and distribute horizontally

display: flex;

In parent container this property have to provide to create layout in one line

align-items:

This property will make elements center of the webpage

Values: center, start and end

justify-content: The justify-content property aligns the flexible container's items when the items do not use all available space on the main-axis (horizontally).

Justify content property values:

  • center: Items are positioned in the center of the container
  • space-between: Items will have space between them
  • space-around: Items will have space before, between, and after them
  • space-evenly: Items will have equal space around

What is flex-wrap CSS?

The flex-wrap CSS property sets whether flex items are forced onto one line or can wrap onto multiple lines. If wrapping is allowed, it sets the direction that lines are stacked.

Values:

  • nowrap: Default value. Specifies that the flexible items will not wrap
  • wrap: Specifies that the flexible items will wrap if necessary
  • wrap-reverse: Specifies that the flexible items will wrap, if necessary, in reverse order

flex-direction:

The flex-direction property in CSS is used to set the direction of the flex container's main axis.

It can have one of four values: row, row-reverse, column, or column-reverse. row is the default value and arranges flex items horizontally.

flex-grow:

The flex-grow CSS property defines how much an element grows relative to other flexible items inside the same parent element. It determines how much space to allocate to an item in a flex parent when the available space exceeds the combined size of all the

It can have values from 0-1, 2, 3...

flex-basis:

Flex-basis is a CSS property used in flexboxes that sets the initial size of a flex item before it grows or shrinks. Flex items exist in flex containers, which grow to fill empty space or shrink to prevent overflow on a web page.

It can values in pixels or rem

14. Z Index

Z-index is the CSS property that controls the stacking order of overlapping elements on a page. An element with a higher z-index value will appear in front of an element with a lower z-index value. The property is called "z-index" because it sets the order of elements along the z-axis

Values: -1, 0, 1

Float property will not work when you will provide for any element absolute position

16. Transform Property

The CSS transform property allows you to rotate, scale, skew, and translate an element. It modifies the coordinate space of the CSS visual formatting model.

CSS transform property allows you to modify an element's shape, size, position, and orientation.

Translate Property in CSS

The translate property allows you to change the position of elements. The translate property defines x- and y-coordinates of an element in 2D

  • translate() - Defines a 2D translation
  • translateX() - Defines a translation, using only the value for the X-axis
  • translateY() - Defines a translation, using only the value for the Y-axis

Values has to be given in pixels

Scale Property

The scale property in CSS resizes an element's width and height in proportion. So, if we have an element that's 100 pixels square, scaling it up by a value of 2 doubles the dimensions to 200 pixels square

  • scale() - Defines a 2D scale transformation
  • scaleX() - Defines a scale by giving a value for the X-axis
  • scaleY() - Defines a scale transformation by giving a value for the Y-axis

Values has to be given in digits ex. 1, 1.2, 1.3, 1.4

Rotate Property

The rotate property allows you to rotate elements. The rotate property defines a value for how much an element is rotated clockwise direction

  • rotate(angle) - Defines a 2D rotation, the angle is specified in the parameter
  • rotateX(angle) - Defines a 2D rotation along the X-axis
  • rotateY(angle) - Defines a 2D rotation along the Y-axis

Values has to be given in degrees for ex. (45deg)

Skew Property

The skew() property is which is used to transform an element in the 2D plane. Skew an element means to pick a point and push or pull it in different directions.

  • skew(angle) - Defines a 2D skew transformation
  • skewX(angle) - Defines a 2D skew transformation along the X-axis
  • skewY(angle) - Defines a 2D skew transformation along the Y-axis

Value has to be given in degrees

18. Animation in CSS

An animation lets an element gradually change from one style to another. (one color to another color) You can change as many CSS properties you want, as many times as you want.

To use CSS animation, you must first specify some keyframes for the animation. Keyframes hold what styles the element will have at certain times.

Animation properties:

  • animation-name
  • animation-duration
  • animation-timing-function
  • animation-delay
  • animation-iteration-count
  • animation-direction - alternate, normal, reverse
@keyframes identifier(animation name) {
    from {
        property: value;
    }
    to {
        property: value;
    }
}

In the example above we have specified when the style will change by using the keywords "from" and "to" (which represents 0% (start) and 100% (complete)).

It is also possible to use percent. By using percent, you can add as many style changes as you like.
